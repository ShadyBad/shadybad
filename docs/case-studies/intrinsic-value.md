# Intrinsic Value Calculator — Case Study

## Problem
Ad‑hoc valuation workflows are inconsistent, hard to audit, and slow to update as markets move.

## Approach
- Live data ingestion via Yahoo Finance and FRED for risk‑free rates
- Parallelized DCF modeling with automated growth/risk adjustments
- Validation and error handling for resilient runs
- Investor‑ready CSV/JSON outputs for dashboards, APIs, and research notebooks

## Impact
- Consistent, auditable valuations with timely market inputs
- Accelerated research loops and simpler sharing across stakeholders
- Drop‑in artifacts for BI and quant tooling

## Tech Highlights
- Python, NumPy/Pandas, parallel execution
- Clean report generation; artifacts tracked for repeatability
